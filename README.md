### UpdateZig 
a scirpt to update Ziglang from offical web, especially master branch.

### Usage
just put the updateZig.sh and the binary of zig together.
```
/zigPath
├── /doc
├── /lib
├── LICENSE
├── updateZig.sh
├── zig
└── zls
```
and cmd:
```
# bash 
./updateZig.sh
```
Than, enjoy your time.
